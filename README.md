# 循迹 - 运动健康分析应用

一个基于 PWA 技术的运动健康分析应用，通过 AI 分析运动截图，帮助你理解身体的内在轨迹。

## 品牌故事

在健康这条路上，我们常常手握两张地图：一张是外在的路径，记录着我们在公园、街道、山野间跑过的每一公里；另一张是内在的风景，描绘着心率、恢复、压力与能量——这些身体默默诉说的无声语言。

"循迹"相信，真正的健康，始于读懂第二张地图。我们的角色，并非教练，而是翻译与向导。

## 功能特点

- 🎨 **精美设计** - 采用独特的自然色调，体现运动、健康、科学的品牌理念
- 📱 **PWA 支持** - 可安装到设备主屏幕，像原生应用一样使用
- 🖼️ **图片上传** - 支持点击上传和拖拽上传运动截图
- 📊 **智能分析** - 模拟 AI 分析，展示恢复状态、压力指数、能量水平等指标
- 📈 **趋势洞察** - 查看运动记录和恢复趋势
- ✨ **流畅动画** - 精心设计的微交互和过渡效果

## 设计理念

### 色彩系统
- 主色调采用自然绿色系，象征健康与生命力
- 使用 oklch 颜色空间，确保色彩感知的准确性
- 柔和的背景色调，营造舒适的使用体验

### 字体排版
- 标题使用 Noto Serif SC，传达专业感和品牌格调
- 正文使用 Noto Sans SC，保证优秀的可读性
- 采用响应式字体大小，适配不同屏幕

### 视觉元素
- 地图网格背景 - 呼应"身体地图"的概念
- 轨迹线条动画 - 象征探索与前进的旅程
- 脉冲圆环 - 体现生命的活力与节奏

## 快速开始

### 本地运行

1. 在 `fitness-tracker` 目录下启动本地服务器
2. 使用浏览器访问 `http://localhost:端口号`

### 使用方式

1. 在欢迎页面阅读品牌故事，点击"开始探索"
2. 在上传页面选择运动截图（支持拖拽）
3. 点击"开始分析"查看 AI 分析结果
4. 在"记录"页面查看运动历史
5. 在"洞察"页面查看趋势分析
